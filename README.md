# Aphotic Group

Placeholder landing page for [aphoticgroup.com](https://aphoticgroup.com).

Static single-page site (`index.html`), hosted on GitHub Pages with a custom apex domain.

## Local preview

Open `index.html` directly in a browser, or serve it:

```sh
python -m http.server 8000
# then visit http://localhost:8000
```

## Deploy

Pushing to the `main` branch publishes automatically via GitHub Pages.
The `CNAME` file pins the custom domain.
